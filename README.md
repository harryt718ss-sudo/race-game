# 🏎️ Python Turbo Racer - 高速賽車挑戰賽

這是一個基於 **Pygame** 開發的復古風格賽車遊戲。遊戲採用偽 3D（2.5D）渲染技術，模擬經典大型電玩賽車（如 OutRun）的極速感，並內建具備智慧追趕機制的 AI 對手。



## 🌟 遊戲特色

* **偽 3D 渲染引擎**：利用數學投影與掃描線算法模擬賽道的深度感與彎道變化，而非單純的 2D 平面移動。
* **動態 AI 對手**：AI 具備「橡皮筋機制」（Rubber Banding），會根據玩家位置動態調整速度：
    * **落後時**：開啟暴怒模式加速追趕。
    * **領先時**：保持全速並頻繁換道干擾玩家。
* **彎道預警輔助**：畫面中心會根據前方 700-800 單位距離處的賽道曲率，顯示輔助提示（`<<<` 或 `>>>`）。
* **氮氣加速系統**：玩家擁有一罐強力氮氣，可於直線段釋放，最高時速可提升近 2 倍。
* **物理路面判定**：區分「賽道」與「草地」，失控衝出賽道會導致車速驟降並喪失抓地力。

## 🎮 操作說明

| 按鍵 | 動作 |
| :--- | :--- |
| **← / →** | 控制賽車左右靈敏轉向 |
| **SPACE (空白鍵)** | 啟動氮氣加速 (Nitro) |
| **ESC** | 立即結束並關閉遊戲 |
